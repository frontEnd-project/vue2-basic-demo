# vue2.0-basic-demo

这里是关于学习vue2.0的相关例子，欢迎一起学习！

**目录说明：**

1. vue-demo

	学习vue过程中，阅读官网文档写出来的，官网给的例子有点简单，所以在这个基础上写了完整的学习例子。

2. vue-examplate

	看了vue文档后，写的比较综合的案例，例如todolist这样的案例。

3. vue-project

	用vue写的一些项目，后续待完善中，已收录如下：

	- build-site-tool 目录下是用ts写的可拖拽的建站工具代码，使用了动态组件，按需加载功能
